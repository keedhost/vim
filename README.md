How to install vim config for Andrii Kondratiev

# 1. Clone repo:
```
git clone git@github.com:keedhost/vim.git ~/.vim
```

# 2. Install FZF:
### For Linux:
```
`git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf && \
~/.fzf/install
```
or
```
sudo pacman -S fzf```
or
```
sudo apt-get install fzf
```
### For macOS:
```
brew install fzf
```

# 3. Install plugins 
vim => :PlugInstall

# 4. Enjoy!
